# Task2Schedule

# Project Overview
task2schedule supports users to make a schedule from a task list. It can help users to schedule as task-driven as possible, and achieve their goals more efficiently.
Also we support Google Calendar integration, so that users can easily sync their schedule to their calendar and manage their time more effectively.

# Features and core functionality
- Goal Setting
- Task & Todo Management
- Schedule Conversion
- Google Calendar Integration
